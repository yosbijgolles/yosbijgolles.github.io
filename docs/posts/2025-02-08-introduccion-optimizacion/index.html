¿Qué es la Optimización Matemática?

La optimización matemática es el proceso de encontrar la mejor solución posible a un problema, sujeto a ciertas restricciones. Es una herramienta fundamental en Operations Research y tiene aplicaciones en logística, finanzas, manufactura, y muchos otros campos.

Componentes de un Problema de Optimización

Todo problema de optimización tiene tres elementos clave:

1. Variables de decisión: Las cantidades que queremos determinar

2. Función objetivo: Lo que queremos maximizar o minimizar

3. Restricciones: Las limitaciones que debemos respetar

Ejemplo Clásico: Programación Lineal

Supongamos que una empresa fabrica dos productos: A y B.

  • Producto A genera $3 de ganancia
  • Producto B genera $4 de ganancia
  • Hay limitaciones de recursos (materias primas, tiempo de máquina)

Queremos maximizar las ganancias respetando las restricciones.

Formulación Matemática

Variables de decisión: - \(x\) = unidades del producto A - \(y\) = unidades del producto B

Función objetivo:

\[ \max \quad z = 3x + 4y \]

Restricciones:

\[ \begin{align} x + 2y &\leq 14 \quad \text{(restricción de material)} \\ 3x - y &\geq 0 \quad \text{(restricción técnica)} \\ x, y &\geq 0 \quad \text{(no negatividad)} \end{align} \]

Resolución con Python

Podemos resolver este problema usando OR-Tools:

from ortools.linear_solver import pywraplp

# Crear el solver
solver = pywraplp.Solver.CreateSolver('GLOP')

# Variables de decisión
x = solver.NumVar(0, solver.infinity(), 'x')
y = solver.NumVar(0, solver.infinity(), 'y')

# Función objetivo: maximizar 3x + 4y
solver.Maximize(3*x + 4*y)

# Restricciones
solver.Add(x + 2*y <= 14)
solver.Add(3*x - y >= 0)

# Resolver
status = solver.Solve()

if status == pywraplp.Solver.OPTIMAL:
    print(f'Solución óptima encontrada:')
    print(f'x = {x.solution_value():.2f}')
    print(f'y = {y.solution_value():.2f}')
    print(f'Ganancia máxima = ${solver.Objective().Value():.2f}')

Salida:

Solución óptima encontrada:
x = 2.00
y = 6.00
Ganancia máxima = $30.00

Tipos de Problemas de Optimización

En este blog exploraremos diferentes tipos:

  • Programación Lineal (LP): Función objetivo y restricciones lineales
  • Programación Entera (IP/MIP): Variables que deben ser enteros
  • Problemas de Ruteo (VRP): Optimización de rutas de vehículos
  • Scheduling: Asignación óptima de recursos en el tiempo
  • Metaheurísticas: Algoritmos aproximados para problemas complejos
